# carefree-cython

`carefree-cython` implemented some commonly used cython functions


## Installation

`carefree-cython` requires Python 3.8 or higher.

```bash
pip install carefree-cython
```

or

```bash
git clone https://github.com/carefree0910/carefree-cython.git
cd carefree-cython
pip install -e .
```
